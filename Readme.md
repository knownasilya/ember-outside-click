
# Ember Click Outside

Click outside component for Ember. Clean and simple.

## Install

```bash
$ ember install nucleartide/ember-click-outside
```

## Use

```hbs
{{#click-outside onClickOutside=(action 'hide')}}
  <p>hide me if you click outside of the element</p>
{{/click-outside}}
```
